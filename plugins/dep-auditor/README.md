# Dependency Auditor

Scans project dependencies for security vulnerabilities, outdated packages, and unused imports.

## Skills

- **audit-deps** — Triggered by: "audit dependencies", "check vulnerabilities", "outdated packages"

## Agents

- **auditor** — Agent that runs audit commands and scans source files

## Install

```
/plugin install dep-auditor@blankdean-claude-plugins
```
